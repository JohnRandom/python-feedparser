entries[i].source

A dictionary with details about the source of the entry.

Comes from

  • /atom10:feed/atom10:entry/atom10:source

entries[i].source.href

The URL of the entry source. The RSS specification states that this must be the URL of an RSS feed, not an HTML page or other feed format.

If this is a relative URI, it is resolved according to a set of rules.

entries[i].source.value

The name of the entry source. The RSS specification states that this should be the title of the RSS feed where this entry came from, not an entry title or page title.
